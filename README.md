
# THU LX计算项目前端工程

## 常用命令行

### 1. 安装和启动本项目
```
cd thu_lx_vue3
npm install
npm run dev
```
### 2. 创建新component组件
```
npm run newpage “模块名称"
```

## Vue 3 + Vite
